# Mobile Banking Management System

## Description

Mobile Banking Management System is an Android application developed in Java that allows users to manage clients, loans, and savings accounts. The application uses SQLite as a local database and follows the MVC (Model-View-Controller) architecture to organize the project structure.

The system provides two types of users:

- **Administrator:** Can manage clients and assign loans.
- **Client:** Can calculate loan payments, manage savings plans, and view/pay their loans.

---

## Features

### Administrator Features

- Administrator login system.
- Client registration.
- View all registered clients.
- Search clients by identification number.
- Assign loans to clients.
- Manage loan information.

### Client Features

- Client login system.
- Calculate loan monthly payments.
- View personal loans.
- Make loan payments.
- Manage savings accounts:
  - Christmas Savings.
  - Education Savings.
  - Vehicle Tax Savings.
  - Extraordinary Savings.
- Update savings amounts.

## Technologies Used

- **Programming Language:** Java
- **Platform:** Android

- **Architecture:** MVC (Model - View - Controller)
- **IDE:** Android Studio

## Installation

### Requirements

- Android Studio installed.
- Android SDK.
- Java Development Kit (JDK).

### Steps

1. Clone the repository:

```bash
git clone https://github.com/SamuelLeal0503/mobile-banking-management-system.git
```

2. Open the project in Android Studio.
3. Wait for Gradle synchronization.
4. Connect an Android device or start an emulator.
5. Run the application.
